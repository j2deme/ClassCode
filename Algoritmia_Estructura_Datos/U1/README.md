# Algoritmia y Estructura de Datos - Unidad 1

En esta unidad se abordan los conceptos fundamentales de la algoritmia y las estructuras de datos. Se explorarán temas como la complejidad algorítmica, tipos de datos básicos, y estructuras de datos lineales y no lineales.

---

## Contenidos

Como parte del análisis de complejidad algorítmica, se revisaron las librerías `time` y `timeit` de Python para la medición de tiempo de ejecución de algoritmos.
